Project Overview

The Student Campus Pulse Dashboard is a functional locally hosted web application developed under the TCS iON Industry Project to analyze and visualize student satisfaction data interactively. The project transforms raw student survey data into meaningful institutional insights through data cleaning, backend API development, and frontend dashboard visualization.

The system helps educational institutions evaluate academic performance, campus facilities, placement confidence, attendance, and student engagement using dynamic filters, KPI cards, charts, and correlation heatmaps.

This project demonstrates a complete workflow from raw Excel survey data to a full-stack analytical dashboard that supports data-driven decision-making.

Setup
Prerequisites:
Python 3.x installed
Jupyter Notebook (for data cleaning notebook)
VS Code or any code editor
Web browser (Chrome recommended)

Required Python Libraries:
Flask
Flask-CORS
Pandas
OpenPyXL

Install Dependencies:
pip install flask flask-cors pandas openpyxl
How to Run

Step 1: Download or Clone the Repository
git clone https://github.com/yourusername/student-campus-pulse-dashboard.git

Step 2: Open Backend Folder
cd student-campus-pulse-dashboard/backend

Step 3: Run Flask Backend
python app.py

Backend will start on:
http://127.0.0.1:5000

Step 4: Open Frontend Dashboard
Navigate to:
frontend/index.html
Open index.html in your browser.

Step 5: Use Dashboard
Apply filters
View KPI cards
Analyze satisfaction charts
Explore heatmap insights
